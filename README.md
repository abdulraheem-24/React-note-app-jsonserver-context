# React Notes App (JSON Server + Context API)

This is a simple Notes Application built using **React**, **Context API**, and **JSON Server** for backend simulation.

## 🚀 Features

* Create notes
* Edit notes
* Delete notes
* Archive notes
* Persistent data using JSON Server

## 🛠️ Technologies Used

* React
* Context API
* Axios
* JSON Server
* Tailwind CSS
* Vite

## 📦 Installation

Clone the repository

git clone https://github.com/abdulraheem-24/React-note-app-jsonserver-context.git

Go to project folder

cd React-note-app-jsonserver-context

Install dependencies

npm install

## ▶️ Run React App

npm run dev

## 🗄️ Run JSON Server

npx json-server --watch db.json --port 3001

## 🌐 API Endpoint

http://localhost:3001/notes
